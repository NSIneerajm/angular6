1) On window machine install the nodejs {version node-v8.11.3-x64.msi} by running this file "node-v8.11.3-x64.msi" download from the url"https://nodejs.org/en/".
2) After install nodejs check the node version and npm version use following command.
	-> node -v (tell the node version)
	-> npm  -v (tell the npm version)
3) Install the angular cli globally.
	-> npm install -g @angular/cli
4) Create the angular project
	->ng new neerajmdemo
	->cd  neerajmdemo
	->ng serve
	
finally run the project on this url :-   "http://localhost:4200/".
